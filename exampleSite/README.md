# Broadsheet Example Site

This directory contains a complete example site demonstrating the Broadsheet theme.

## Running Locally

```bash
cd hugo-broadsheet/exampleSite
hugo server --themesDir /path/to/parent/of/hugo-broadsheet
```

## Contents

- `hugo.toml` - Example site configuration
- `content/about.md` - Sample About page
- `content/contact.md` - Sample Contact page
- `content/posts/` - Sample articles demonstrating different content types

## Using as a Starting Point

Copy these files to your own Hugo site as a starting point:

1. Copy `content/` files to your site's `content/` directory
2. Adapt `hugo.toml` settings for your site
3. Edit the sample pages with your own content
