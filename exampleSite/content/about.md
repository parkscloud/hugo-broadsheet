Welcome to The Daily Broadsheet, a demonstration site for the [Broadsheet Hugo theme](https://github.com/parkscloud/hugo-broadsheet).

## About This Site

This is an example site showing how to use the Broadsheet theme. You can use this as a starting point for your own news site, magazine, or blog.

## About Broadsheet

Broadsheet is a newspaper-style theme for Hugo designed for:

- **News sites** covering current events
- **Magazines** with featured articles
- **Blogs** that want a classic editorial look

The theme features a traditional masthead, multi-column layouts, and clean typography optimized for reading.

## Getting Started

1. Copy the files from this `exampleSite/` directory to your Hugo site
2. Edit `hugo.toml` with your site details
3. Replace the sample posts with your own content
4. Customize the About page (this file!)

For full documentation, see the [README](https://github.com/parkscloud/hugo-broadsheet).
